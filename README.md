# Botcamp XP Inc. - Full Stack Developer

## Desafio de Projeto: Blindando Seu Código com TDD e Testes Unitários Usando .NET Core

### Descrição
Este projeto tem como objetivo demonstrar a aplicação de Test-Driven Development (TDD) e testes unitários na criação de uma calculadora utilizando .NET Core. O desafio é parte do treinamento de desenvolvedores Full Stack da Botcamp XP Inc.

### Requisitos
- .NET Core 3.1 ou superior
- Ferramentas para desenvolvimento de testes unitários (por exemplo, xUnit, NUnit)

### Funcionalidades
- Implementação de operações básicas da calculadora, como adição, subtração, multiplicação e divisão.
- Implementação de testes unitários para garantir a precisão e robustez das operações.
